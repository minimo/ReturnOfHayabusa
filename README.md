# Returun-of-Hayabusa
## はやぶさの帰還

７００万Kmを旅して、イトカワのサンプルを地球へ持ち帰ってください。
隕石と宇宙放射線を避けつつ、地球からイトカワ、イトカワから地球へはやぶさを導いてください。

＜＜操作方法＞＞
PCの場合：カーソルキーで移動。
スマートフォンの場合：画面の上下左右の縁をタッチで移動。

移動の際には推進剤（Propellant）を消費します。
慣性が働いている為、減速又は止まるには逆方向に移動する必要があります。
推進剤の量には限りがある為、なるべく温存して進んでください。
推進剤が０になると帰還不能でゲームオーバーとなります。

宇宙放射線（☆）、隕石（大小）に当たるとシールドが減少します。
シールドは徐々に回復していきますが、シールドが少ない状態で放射線に当たるとイオンエンジンが停止します。
４つエンジンが停止すると帰還不能でゲームオーバーとなります。

たまに遭難しているクマさんがいますので助けてあげてください。
お礼に推進剤とシールドの回復をしてくれます。
